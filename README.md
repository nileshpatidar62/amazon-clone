 # Amazon Clone Website 🛒

This project showcases a modern e-commerce website interface by recreating the **Amazon homepage** using **HTML and CSS**. It demonstrates practical front-end development skills, responsive web design principles, and clean UI implementation without using JavaScript or external frameworks.

## 🎯 Purpose of the Project

**Portfolio:** A hands-on project showcasing the ability to build a professional and responsive e-commerce website interface.

**Learning:** Gained practical experience in HTML, CSS, Flexbox, CSS Grid, responsive layouts, and modern UI design techniques.

**Placement Ready:** Demonstrates strong front-end development fundamentals and the ability to recreate real-world commercial websites following clean coding practices.

---

## 🛠️ Tech Stack

**Frontend:** HTML5, CSS3

**Layout:** Flexbox, CSS Grid

**Design:** Responsive Web Design, Media Queries

**Development Tools:** Visual Studio Code, Git, GitHub

---

## 🔄 Development Workflow

The project follows a structured front-end development workflow to build a responsive and user-friendly e-commerce interface.

### HTML Structure

Developed a semantic and well-organized webpage structure using HTML.

**Steps:**

* Created reusable page sections including Navbar, Hero Banner, Product Categories, Shopping Panels, and Footer.
* Used semantic HTML elements to improve accessibility and maintainability.
* Organized content into logical sections for better readability and scalability.

### CSS Styling

Applied modern CSS techniques to create a responsive and visually appealing interface.

**Steps:**

* Designed responsive layouts using Flexbox and CSS Grid.
* Implemented hover effects, transitions, spacing, typography, and color styling.
* Used media queries to optimize the website for desktop, tablet, and mobile devices.
* Maintained consistent UI components across the entire webpage.

---

## 🌐 Website Features

The website recreates the visual appearance of Amazon's homepage while emphasizing responsive design and clean user interface development.

**Implemented Features:**

* Responsive navigation bar with Amazon logo, search bar, delivery location, account options, returns section, and shopping cart.
* Hero banner with promotional images.
* Product category cards displaying images, titles, and shopping links.
* Responsive product grid layout for different screen sizes.
* Multi-section footer containing useful navigation links and company information.
* Interactive hover animations for buttons, cards, and navigation elements.
* Fully responsive design compatible with desktop, tablet, and mobile devices.
